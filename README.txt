City Builder

What it is:
- Base Project for an isometric city/ builder game
- All assets were made in aseprite
- Sprites are 64x64. Tiles are 32 x 64 (half the spite)
- Test placements can be done by pressing 1 or 2 and clicking on the tile.

Functionality:

- Panning camera
- Zooming in and out
- clicking tiles returns X and Y coordinates 
- Placeables can be adjusted/extended in control.java
